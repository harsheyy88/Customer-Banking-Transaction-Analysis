# Customer Banking Transaction Analysis

Exploratory data analysis of customer banking transactions using Python and Power BI to uncover spending patterns, fraud activity, and transaction trends.

## Project Overview
This project analyzes banking transaction data to identify customer behavior, transaction channels, and fraud patterns.

## Tools Used
- Python
- Power BI
- Pandas
- numpy
- Data Visualization

## Key Insights
- UPI transactions have the highest transaction volume.
- Debit transactions are more frequent than credit transactions.
- Fraud transactions account for a small percentage of total transactions.

## Power BI Dashboard
The Power BI dashboard visualizes transaction trends, fraud detection, and channel performance.
